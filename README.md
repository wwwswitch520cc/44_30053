<p>身处一个充满谜题与冒险的 3D 世界，随着遇到的 Mote 逐步提升难度，巧妙而富有逻辑性的关卡设计让玩家得以扮演 Lumote 的角色获得放松，同时令 Lumote 与世界的连系得以舒缓。整个世界和每个谜题随时一目了然。</p>
<p>由于 Great Depth 通过音乐来向你告知关键情境和 Lumote 的情感状态，因此令人惊叹的配乐将伴随你踏上宏伟的电子音乐之旅。</p>
